# Jogo Sudoku
Este é um projeto simples de Sudoku implementado em Java. O jogo permite ao usuário interagir com um tabuleiro de Sudoku, preencher números, verificar o status do jogo, remover números e finalizar o jogo.

## Como usar

1. Clone o repositório:
   ```shell
   git clone https://github.com/ppablolds/Jogo_Sudoku.git
   ```
2. Compile o código:
   ```shell
   javac Main.java
   ```
3. Execute o jogo com os argumentos que representam o tabuleiro:
   ```shell
   java Main "0,0;5,true" "0,1;3,true" "0,4;7,true" "1,0;6,true" "1,3;1,true" "1,4;9,true" "1,5;5,true" "2,1;9,true" "2,2;8,true" "2,7;6,true" "3,0;8,true" "3,4;6,true" "3,8;3,true" "4,0;4,true" "4,3;8,true" "4,5;3,true" "4,8;1,true" "5,0;7,true" "5,4;2,true" "5,8;6,true" "6,1;6,true" "6,6;2,true" "6,7;8,true" "7,3;4,true" "7,4;1,true" "7,5;9,true" "7,8;5,true" "8,4;8,true" "8,7;7,true" "8,8;9,true"
   ```

<hr />

### Comandos disponíveis:

- 1: Iniciar um novo jogo.
- 2: Colocar um número no tabuleiro.
- 3: Remover um número.
- 4: Visualizar o estado atual do jogo.
- 5: Verificar status do jogo (se o jogo tem erros ou está completo).
- 6: Limpar o jogo.
- 7: Finalizar o jogo.
- 8: Sair.
